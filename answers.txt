Q1: Why might we consider using Git?

Answer: It allows us to track changes, collaborate with others, manage different versions of a project, and recover previous versions if need be.


Q2: What field could data science be applied to? Give an Example.

Q3: What is the difference between Merging and Rebasing?

Answer: Merging combines changes from two branches and creates a merge commit, while rebasing moves a branch's commits onto another branch to create a cleaner project history.
